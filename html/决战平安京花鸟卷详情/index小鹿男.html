小鹿男

日文CV：早见沙织

中文CV：杨婧

位置 : 侍

特长 : 持续输出/生存

推荐路线 : 恢复/输出

难度 :

使用技巧 : 利用远野之怒的消耗对手，并通过林间踏风的加速效果风筝或者追击，森林守护提供护盾的同时在结束时可以对周围敌人造成伤害。团战中除了可以用来入场，释放得当还能起到分割对方团队站位的作用。

自强

小鹿男周围1000码内有任何单位被封印都会使自己获得强化提高小鹿男最多 (20%技能伤害)。 每个非式神单位被封印可获得 (5%强化), 式神单位被封印可直接获得 20%强化 。一段时间后强化会逐渐衰减。

林间踏风

小鹿男向前奔跑获得 60%移速加成并在3秒内逐渐衰减，期间将对撞到的敌人造成 90 和2.5秒的55%减速效果。

远野之怒

小鹿男持续4.5秒在周围不断引发总计15次随机爆炸，每次爆炸对命中的敌人造成8 (+8%额外攻击) (+1.25%目标最大生命值) >点 物理伤害。，该伤害对野怪有上限。

森林守护

小鹿男张开一个可吸收20 (+7%最大生命值) 点伤害的护盾，持续3秒。护盾结束时触发爆炸对自身周围敌人造成20 (+6%目标最大生命值)点 物理伤害（对野怪伤害上限为500点）。

鹿影奔腾

小鹿男向指定方向发起冲锋，对途中撞到的敌人造成180 (+120%额外攻击) 点 物理伤害点物理伤害并将其向两侧击飞。

瞬步/盾守

瞬步

瞬间消失然后出现在目标方位

盾守

保护自己和附近一名生命垂危的友方式神，分别生成一个可吸收125(+25*当前等级) 点伤害的护盾，持续3秒。15秒内再次受到盾守效果将减半。
